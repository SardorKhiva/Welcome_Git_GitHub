Git bilan ishlovchi asosiy buyruqlar

  1. Git repository yaratish

    Git repository yaratish uchun quyidagi buyruqdan foydalanamiz:

    git init

    Bu buyruq joriy papkada yangi Git repository yaratadi.

  2. Git repositoryga fayl qo'shish

    Faylni Git repositoryga qo'shish uchun quyidagi buyruqdan foydalanamiz:

    git add

    Bu buyruq faylni staging area ga qo'shadi.

  3. Git repositorydagi o'zgarishlarni saqlash

    O'zgarishlarni saqlash uchun quyidagi buyruqdan foydalanamiz:

    git commit -m "O'zgarishlar haqida qisqacha ma'lumot"

    Bu buyruq staging area dagi o'zgarishlarni Git repositoryga saqlaydi.

  4. Git repositorydagi o'zgarishlarni ko'rish

    O'zgarishlarni ko'rish uchun quyidagi buyruqdan foydalanamiz:

    git status

    Bu buyruq joriy holatni va staging area dagi o'zgarishlarni ko'rsatadi.

  5. Git repositorydagi tarixni ko'rish

    Git repositorydagi tarixni ko'rish uchun quyidagi buyruqdan foydalanamiz:

    git log

    Bu buyruq barcha commitlarni va ularning tafsilotlarini ko'rsatadi.

  6. Git repositorydan faylni o'chirish

    Faylni Git repositorydan o'chirish uchun quyidagi buyruqdan foydalanamiz:

    git rm

    Bu buyruq faylni staging area dan o'chiradi va keyingi commitda o'chiriladi.

  7. Git repositoryni boshqa joyga nusxalash

    Git repositoryni boshqa joyga nusxalash uchun quyidagi buyruqdan foydalanamiz:

    git clone

    Bu buyruq berilgan URL manzilidagi Git repositoryni joriy papkaga nusxalaydi.

  8. Git branch yaratish

    Yangi branch yaratish uchun quyidagi buyruqdan foydalanamiz:

    git branch

    Bu buyruq yangi branch yaratadi, lekin uni faollashtirmaydi.

  9. Git branchga o'tish

    Yaratilgan branchga o'tish uchun quyidagi buyruqdan foydalanamiz:

    git checkout 
    git switch

    Bu buyruq berilgan branchga o'tadi.

  10. Git branchlarni ko'rish

    Yaratilgan branchlarni ko'rish uchun quyidagi buyruqdan foydalanamiz:

    git branch

    Bu buyruq barcha branchlarni va joriy branchni ko'rsatadi.

  11. Git branchni o'chirish

    Branchni o'chirish uchun quyidagi buyruqdan foydalanamiz:

    git branch -d

    Bu buyruq berilgan branchni o'chiradi, agar u bo'sh bo'lsa.

  12. Git remote repositoryga push qilish

    O'zgarishlarni remote repositoryga push qilish uchun quyidagi buyruqdan foydalanamiz:

    git push origin

    Bu buyruq joriy branchdagi o'zgarishlarni remote repositoryga yuboradi.

  13. Git remote repositorydan pull qilish

    Remote repositorydan o'zgarishlarni olish uchun quyidagi buyruqdan foydalanamiz:

    git pull origin

    Bu buyruq remote repositorydagi o'zgarishlarni joriy branchga qo'shadi.

  14. Git remote repositoryni qo'shish

    Remote repositoryni qo'shish uchun quyidagi buyruqdan foydalanamiz:

    git remote add origin repository_url

    Bu buyruq remote repositoryni joriy Git repositoryga qo'shadi.

  15. Git remote repositoryni ko'rish

    Remote repositorylarni ko'rish uchun quyidagi buyruqdan foydalanamiz:

    git remote -v

    Bu buyruq barcha remote repositorylarni va ularning URL manzillarini ko'rsatadi.

  16. Git merge qilish

    Branchlarni birlashtirish uchun quyidagi buyruqdan foydalanamiz:

    git merge

    Bu buyruq berilgan branchdagi o'zgarishlarni joriy branchga qo'shadi.

  17. Git stash qilish

    Joriy o'zgarishlarni vaqtincha saqlash uchun quyidagi buyruqdan foydalanamiz:

    git stash

    Bu buyruq joriy o'zgarishlarni stashga saqlaydi va repositoryni toza holatga qaytaradi.

  18. Git stashdan o'zgarishlarni qaytarish

    Stashdagi o'zgarishlarni qaytarish uchun quyidagi buyruqdan foydalanamiz:

    git stash pop

    Bu buyruq stashdagi oxirgi o'zgarishlarni joriy branchga qo'shadi.

  19. Git stashdagi o'zgarishlarni ko'rish

    Stashdagi o'zgarishlarni ko'rish uchun quyidagi buyruqdan foydalanamiz:

    git stash list

    Bu buyruq stashdagi barcha o'zgarishlarni ro'yxatini ko'rsatadi.

  20. Git stashdagi o'zgarishlarni o'chirish

    Stashdagi o'zgarishlarni o'chirish uchun quyidagi buyruqdan foydalanamiz:

    git stash drop

    Bu buyruq stashdagi oxirgi o'zgarishni o'chiradi.

  21. Git stashdagi o'zgarishlarni tozalash

    Barcha stashdagi o'zgarishlarni tozalash uchun quyidagi buyruqdan foydalanamiz:

    git stash clear

    Bu buyruq stashdagi barcha o'zgarishlarni o'chiradi.

  22. Git konfiguratsiyasini ko'rish

    Git konfiguratsiyasini ko'rish uchun quyidagi buyruqdan foydalanamiz:

    git config --list

    Bu buyruq Git konfiguratsiyasidagi barcha sozlamalarni ko'rsatadi.

  23. Git konfiguratsiyasini o'zgartirish

    Git konfiguratsiyasini o'zgartirish uchun quyidagi buyruqdan foydalanamiz:

    git config --global user.name "Ismingiz"
    git config --global user.email "Emailingiz"

    Bu buyruqlar Git konfiguratsiyasida foydalanuvchi nomi va email manzilini o'rnatadi.

  24. Git konfiguratsiyasini o'chirish

    Git konfiguratsiyasini o'chirish uchun quyidagi buyruqdan foydalanamiz:

    git config --global --unset user.name
    git config --global --unset user.email

    Bu buyruqlar Git konfiguratsiyasidan foydalanuvchi nomi va email manzilini o'chiradi.

  25. Git konfiguratsiyasini faylga yozish

    Git konfiguratsiyasini faylga yozish uchun quyidagi buyruqdan foydalanamiz:

    git config --global -e

    Bu buyruq Git konfiguratsiya faylini ochadi va uni tahrirlash imkonini beradi.

  26. Git konfiguratsiyasini fayldan o'qish

    Git konfiguratsiyasini fayldan o'qish uchun quyidagi buyruqdan foydalanamiz:

    git config --global -l

    Bu buyruq Git konfiguratsiya faylidan barcha sozlamalarni ko'rsatadi.

  27. Git konfiguratsiyasini tozalash

    Git konfiguratsiyasini tozalash uchun quyidagi buyruqdan foydalanamiz:

    git config --global --unset-all

    Bu buyruq Git konfiguratsiyasidagi barcha sozlamalarni o'chiradi.

  28. Git konfiguratsiyasini faylga yozish

    Git konfiguratsiyasini faylga yozish uchun quyidagi buyruqdan foydalanamiz:

    git config --global -e

    Bu buyruq Git konfiguratsiya faylini ochadi va uni tahrirlash imkonini beradi.

  29. Git konfiguratsiyasini fayldan o'qish

    Git konfiguratsiyasini fayldan o'qish uchun quyidagi buyruqdan foydalanamiz:

    git config --global -l

    Bu buyruq Git konfiguratsiya faylidan barcha sozlamalarni ko'rsatadi.
